Multilingual Clinical Support

Ask a clinical question in any language and review a translated answer you can check.

This page is for multilingual case communication. Add your own OpenRouter API key, enter the case question, optionally attach images, and run the analysis.

Disclaimer: This is a communication aid, not a diagnostic tool. Clinical decisions should always be reviewed by a qualified clinician. Do not paste directly identifying patient information.

Before you start

This page requires your own OpenRouter API key. The key is forwarded only with your request and is not stored by the server.

The answer view appears after you submit. Advanced confidence review is available separately and does not need to be opened for routine use.
